  • 每条变色龙恰有一个自己的爱慕对象，保证任何变色龙的爱慕对象都是异性且两人原色不同。爱慕关系是单向的。

然而你既不知道变色龙们的雌雄，也不知道它们的原色，也不知道它们爱慕的对象，但你可以举行 次见面会：

由你指定的一个集合中的变色龙将聚在一起，如果一条变色龙爱慕的对象在场，它的颜色就会变成对方的原色，否则将保持自己的颜色。见面会时你可以数出所有参加见面会的变色龙的颜色数。

你的目标是猜出所有具有相同原色的变色龙对。

int tot = 0;
void SOLVE(vector<int> &lis, int x) {
    if (lis.size() == 1) return tot++, add_E(lis[0], x);
    int mid = lis.size() >> 1;
    vector<int> Llis(lis.begin(), lis.begin() + mid),
                Rlis(lis.begin() + mid, lis.end());
    if (ask(Llis, x)) {
        SOLVE(Llis, x);
        if (tot != 3 && ask(Rlis, x)) SOLVE(Rlis, x);
    }
    else SOLVE(Rlis, x);
}
